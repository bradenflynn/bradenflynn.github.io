Project Case Study

Toyota Price Prediction Model

R Programming Linear Regression SQL Data Mining

01. Project Overview

This analysis focuses on predicting used car prices for Toyota vehicles using a dataset of 6,738 records. The goal was to identify the key drivers of vehicle value to help auto dealers and private sellers estimate market prices accurately.

Dataset Scope

6,738 independent vehicle records across multiple Toyota models (Supra, Land Cruiser, RAV4, etc.) from 2012–2020.

Core Challenge

Accounting for massive price variability (e.g., Land Cruiser vs. Aygo) while maintaining predictive accuracy using linear regression.

02. Methodology & Analysis

Data Engineering

The process began with rigorous data cleaning, addressing outliers in MPG values (some records showed >200 MPG) and imputing missing values to preserve the full dataset. Exploratory analysis was conducted using sqldf for complex queries and ggplot2 for visualization.

Statistical Approach

A correlation matrix was used to isolate variables with the strongest relationship to price. Engine size (+0.73) and vehicle age (+0.42) were identified as the primary drivers, while mileage showed a strong negative correlation (-0.73) as expected.

03. Key Insights & Results

Regression Performance

The linear regression model achieved an R² of 0.53, meaning 53% of the price variability is explained solely by engine size.
